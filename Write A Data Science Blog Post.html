def drop_row(df):
    missing_rows = (df.isnull().sum(axis=1))
    df = df[missing_rows<1]
    return df

I arranged data set for my questions and I used only relevant feature for each questions. Also, I dropped the rows that have missing valued for this subset of data.

def region(df):
    europe = [ 'United Kingdom', 'Germany', 'France','Netherlands', 'Iceland', 'Norway','Denmark','Ireland', 
          'Sweden', 'Luxembourg', 'Switzerland', 'Austria','Finland',
          'Belgium', 'Liechtenstein', 'Spain', 'Italy','Poland']

    america = ['United States', 'Canada']

    far_east = ['India', 'China', 'Japan', 'Thailand', 'Singapore', 'Hong Kong', 'South Korea', 'Taiwan', 'Malaysia',
            'Indonesia', 'Vietnam']
    
    df['Region']= df['Country'].apply(lambda x: 'Europe' if x in europe else ('America' if x in america
                                           else ('Far East'if x in far_east else 'Other')))
    return df

Bar chart based on country and salary shows no meaningful insight therefore I groupped the country according to their region with 'region' function.
